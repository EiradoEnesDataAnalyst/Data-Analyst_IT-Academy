## 🧠 What Makes This Project Stand Out

* 🌐 **Global-scale analysis** across countries and income groups
* 📊 **Fully interactive visualizations** (not static slides)
* 🧮 **Advanced statistical modeling & ML insights**
* 🗺️ **3D geospatial exploration** of energy distribution
* 📈 **Clear narrative storytelling** backed by data

---

## ⚙️ Tech Stack

This project combines a powerful and modern data stack:

* 🐍 **Python Ecosystem** → Pandas, NumPy
* 📓 **Jupyter Notebooks** → Exploration & development
* 📊 **Plotly** → Interactive visual storytelling
* 🗺️ **Mapbox API** → Advanced geospatial visualization
* 🤖 **Scikit-learn & Statsmodels** → Machine learning & statistical analysis
* 📝 **Quarto** → Report generation
* 🚀 **GitHub Pages** → Deployment

---

## 📡 Data Sources

High-quality, trusted datasets power this analysis:

* 🌍 **Our World in Data** (Oxford Martin School)
* ⚡ **Ember** — Global electricity insights
* 💰 **World Bank** — Economic indicators
* ☀️ **IRENA** — Renewable energy data
